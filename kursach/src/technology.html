Используемые технологии

В данном разделе описаны основные технологии, используемые на сайте.

Bootstrap 4

Bootstrap (также известен как Twitter Bootstrap) — свободный набор инструментов для создания сайтов и веб-приложений. Включает в себя HTML- и CSS-шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейса, включая JavaScript-расширения.

CanvasJS

CanvasJS редставляет из себя отзывчивую библиотеку HTML5 диаграмм с прекрасной оптимизацией, простой API и совместимостью с современными браузерами

Popper.js

Popper.js предоставляет максимум возможностей для динамически всплывающих подсказок на сайте. Плагин самостоятельно определяет позиционирование элементов и благодаря своему алгоритму высчитывает координаты, где будет появляться подсказка. Также, вся работа производится без какой либо зависимости от внешних библиотек.
